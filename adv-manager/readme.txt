Инструкция по запуску приложения:

1. Поместить файлик credentials.properties (шаблон) в безопасное место файловой системы,
 доступное на чтение/запись только нужным юзерам (юзеру)
2. В копии файлика credentials.properties (в безопасном месте)
 заполнить 5 необходимых полей (см. файлик)
3. запустить приложение с VM Option
-Dsecrets.file.path=<полный путь к файлику, включая имя>
пример: -Dsecrets.file.path="C:\users\adwords\credentials.properties"
